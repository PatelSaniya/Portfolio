# Saniya's Ramen Portfolio

An interactive 3D portfolio for Saniya, built with Three.js and Webpack. The experience uses a ramen-shop inspired scene to present Saniya's resume, projects, technical skills, credits, and contact details in a playful recruiter-friendly format.

## Features

- 3D ramen-shop portfolio scene
- Project capsule menu with detailed project views
- Technical skills arcade screen
- Resume and about screens for Saniya
- Credits board and interactive navigation

## Tech Stack

- Three.js
- Webpack
- GSAP
- Howler
- JavaScript, HTML, CSS

## Setup

Install [Node.js](https://nodejs.org/en/download/), then run:

``` bash
npm install
```

Start the local development server at `localhost:8080`:

``` bash
npm run dev
```

Create a production build in `dist/`:

``` bash
npm run build
```
